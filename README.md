# Achar Business in Nepal

A comprehensive list of pickle (achar) businesses in Nepal. This repository aims to document and promote local achar manufacturers and sellers across Nepal.

## About Achar

Achar (pickle) is an integral part of Nepali cuisine. These spicy, tangy condiments are made from various fruits, vegetables, and spices, and are served alongside main meals to enhance flavor.

## Popular Nepali Achar Types

1. **Mango Achar (Aap ko Achar)** - Sweet and spicy mango pickle
2. **Gundruk** - Fermented leafy vegetable
3. **Dalle Khursani Achar** - Hot chili pickle
4. **Lapsi Achar** - Hog plum pickle
5. **Timur Achar** - Sichuan pepper pickle
6. **Nimbu Achar** - Lemon pickle
7. **Amla Achar** - Indian gooseberry pickle
8. **Tama Achar** - Bamboo shoot pickle
9. **Golbheda Achar** - Tomato pickle
10. **Mixed Vegetable Achar** - Combination of vegetables
